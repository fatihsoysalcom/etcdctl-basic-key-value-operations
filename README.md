# Etcdctl Basic Key Value Operations

This bash script demonstrates fundamental etcdctl commands for interacting with etcd. It covers setting, getting, deleting, and managing keys with TTL, illustrating etcd's key-value store capabilities.

## Language

`bash`

## How to Run

1. Ensure etcd is running and accessible.
2. Save the code as `etcdctl_example.sh`.
3. Make it executable: `chmod +x etcdctl_example.sh`.
4. Run it: `./etcdctl_example.sh`.
